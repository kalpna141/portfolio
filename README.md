# Kalpna Thakur — Portfolio

Personal portfolio site built with React, Vite, Tailwind CSS, and Framer Motion.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Editing content

All resume content (skills, experience, projects, education, contact info) lives in
`src/data/portfolioData.js` — edit that file to update the site.
